## 📱 소음제로 앱 정보

**소음제로**는 생활 소음 문제를 과학적으로 해결하는 스마트 측정 및 신고 앱입니다.

### 주요 기능
- 📊 실시간 소음(dB) 측정 및 데이터 저장
- 🎥 소음 현장 동영상 녹화
- 🚗 차량 번호판 자동 인식 (ML Kit + Tesseract OCR)
- 📄 신고용 PDF 리포트 자동 생성
- ☁️ 오프라인 우선 + 클라우드 동기화
