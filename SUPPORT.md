## ❓ 자주 묻는 질문 (FAQ)

### 1. 앱 사용법

**Q: 소음을 측정하려면 어떻게 하나요?**
A: 메인 화면에서 '측정 시작' 버튼을 누르면 실시간으로 소음(dB)이 측정됩니다. 측정 데이터는 자동으로 저장됩니다.

**Q: 번호판 인식은 어떻게 사용하나요?**
A: 측정 중 카메라 아이콘을 탭하면 번호판 촬영 모드로 전환됩니다. 차량 번호판을 카메라로 비추면 자동으로 인식됩니다.

**Q: 신고서는 어떻게 생성하나요?**
A: 측정 완료 후 '리포트 생성' 버튼을 누르면 PDF 형식의 신고서가 자동으로 생성됩니다. 이메일이나 다른 앱으로 공유할 수 있습니다.

### 2. 개인정보 및 데이터

**Q: 수집된 데이터는 어떻게 사용되나요?**
A: 소음 측정 데이터와 위치 정보는 오직 신고 목적으로만 사용됩니다. 사용자의 동의 없이 제3자에게 공유되지 않습니다. 자세한 내용은 [개인정보 처리방침](privacy.html)을 참고해주세요.

**Q: 내 데이터를 삭제할 수 있나요?**
A: 네, 가능합니다. 앱 내 '설정' → '데이터 관리'에서 개별 데이터를 삭제하거나, 회원 탈퇴 시 모든 데이터가 완전히 삭제됩니다.

**Q: 오프라인에서도 사용할 수 있나요?**
A: 네, 소음 측정과 데이터 저장은 오프라인에서도 가능합니다. 클라우드 동기화와 일부 기능은 인터넷 연결이 필요합니다.

### 3. 기술적 문제

**Q: 앱이 실행되지 않아요.**
A:
1. 앱을 완전히 종료 후 재실행해보세요
2. 기기를 재부팅해보세요
3. 앱을 삭제 후 재설치해보세요
4. 문제가 지속되면 고객센터로 문의해주세요

**Q: 소음 측정이 부정확한 것 같아요.**
A:
1. 마이크 권한이 허용되어 있는지 확인해주세요
2. 기기의 마이크 상태를 점검해주세요
3. 주변 소음이 너무 크거나 작지 않은지 확인해주세요
4. 앱을 재시작해보세요

**Q: 번호판 인식이 잘 안 돼요.**
A:
1. 카메라 권한이 허용되어 있는지 확인해주세요
2. 번호판이 선명하게 보이도록 거리와 각도를 조절해주세요
3. 조명이 적절한지 확인해주세요 (너무 어둡거나 밝으면 인식률이 떨어집니다)
4. 번호판이 가려지지 않았는지 확인해주세요

## 🔐 개인정보 보호

소음제로는 사용자의 개인정보 보호를 최우선으로 생각합니다.

### 수집하는 정보
- 회원정보: 이메일 주소, 비밀번호
- 측정 데이터: 소음 수치(dB), 측정 시간
- 위치 정보: GPS 좌표, 주소
- 영상 정보: 카메라 촬영 영상, 번호판 정보

### 개인정보 관련 권리
정보주체는 언제든지 다음 권리를 행사할 수 있습니다:
1. 개인정보 열람 요구
2. 오류 정정 요구
3. 삭제 요구
4. 처리정지 요구

자세한 내용은 [개인정보 처리방침](privacy.html)을 참고해주세요.
